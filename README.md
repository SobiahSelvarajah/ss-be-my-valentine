# 💕 Valentine Website (Next.js)

A sweet single-page Valentine website built with **Next.js (App Router)** and **Tailwind CSS**.

This site was created to ask an important question in the most on-brand way possible:  
**with code, teasing buttons, and absolutely no spoilers.**

---

## Features

- Single-page Valentine experience
- Playful **“No” button** that dodges the cursor
- **Spoiler-free** final screen (the plan is a surprise!)
- Fake **“Reveal the plan”** button that refuses with a laughing cat meme
- Responsive, mobile-friendly design
- Built with modern Next.js best practices

---

## 🛠️ Tech Stack

- **Next.js** (App Router)
- **React**
- **Tailwind CSS**
- Vanilla JavaScript (no TypeScript)
